# Identity-Security-Platform
Plataforma self-hosted de identidade, autenticação e gerenciamento de segredos baseada em Keycloak, Infisical e Docker. Inclui SSO corporativo, MFA, reverse proxy seguro, integração com Microsoft 365, Cloudflare Tunnel, observabilidade, backups e arquitetura segmentada focada em segurança e escalabilidade.
